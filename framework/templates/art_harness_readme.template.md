# Art Harness

This folder owns art direction, visual references, asset briefs, art reviews, and art work items for this project.

## Folder Map

```text
art/
  direction/
  concepts/
  characters/
  gameplay_objects/
  environment/
  animation_vfx/
  production/
  reviews/
  work_items/
```

## Image Stages

| Stage | Use |
|---|---|
| `concept_reference` | Visual concept review |
| `modeling_reference` | Orthographic or part-focused 3D modeling references |
| `production_reference` | Approved production references |
| `legacy` | Historical references |
| `superseded_reference` | Replaced or rejected references |

## Principles

- Art should support gameplay readability first.
- Generated images are review references, not final assets by default.
- Every asset name must map to a clear production definition.
- If an image no longer matches the target asset, mark it as superseded or move it to legacy.
