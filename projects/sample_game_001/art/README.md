# Sample Art Harness

This folder demonstrates how a project can organize art direction, concept references, asset briefs, reviews, and art work items.

## Folder Map

| Folder | Purpose |
|---|---|
| `direction/` | Overall visual rules |
| `concepts/` | Concept brief and generated image reviews |
| `characters/` | Character design references |
| `gameplay_objects/` | Collectibles, hazards, goals, and interactive objects |
| `environment/` | Level and background art direction |
| `animation_vfx/` | Motion and effect direction |
| `production/` | Asset briefs and production strategy |
| `reviews/` | Art review decisions |
| `work_items/` | Art-specific tasks |
