# 세로 단면: 첫 플레이 가능 버전

상태: 초안
대상 버전: `v0.2_first_playable`

## 의미

첫 플레이 가능 버전은 기능 하나가 아니라 게임 컨셉의 핵심 경험을 아주 짧게 플레이 가능한 형태로 증명하는 세로 단면이다.

## 플레이어에게 약속하는 경험

미정

예시 형식:

```text
플레이어는 짧은 테스트 레벨에서 핵심 조작을 사용해 하나의 문제를 해결하고 목표 지점에 도달한다.
```

## 필요한 기획 입력

- 선택된 게임 컨셉
- 게임 설계 원칙
- 핵심 반복 구조
- 첫 플레이 가능 버전의 시스템 구성요소

## 필요한 제작 산출물

| 분야 | 필요한 산출물 | 상태 |
|---|---|---|
| 기획 | 첫 반복 구조의 규칙과 수치 | 미정 |
| 아트 | 가독성 있는 임시 에셋 | 미정 |
| 개발 | 플레이 가능한 조작/시스템 기본 구조 | 미정 |
| UX | 행동, 위험, 보상, 목표에 대한 피드백 | 미정 |
| QA | 경로 검증 시나리오 | 미정 |
| 레벨 | 짧은 테스트 경로 | 미정 |

## 검증 기준

이 세로 단면은 아래가 충족되어야 완료다.

1. 플레이어가 시작할 수 있다.
2. 플레이어가 핵심 행동을 수행할 수 있다.
3. 플레이어가 하나의 도전 또는 판단 상황을 만난다.
4. 플레이어가 피드백, 보상, 진행 중 하나를 받는다.
5. 플레이어가 목표 조건에 도달한다.
6. 플레이테스트 또는 자동화 근거가 기록된다.

## 하지 않을 범위

- 최종 아트
- 많은 콘텐츠 양
- 장기 성장 구조
- 전체 저장 시스템
- 릴리즈 패키징
