# Sample Core Loop System

## Purpose

Define the repeatable loop that the first playable version must prove.

## Loop

1. Start the scenario.
2. Read the goal.
3. Perform the core action.
4. Face a challenge.
5. Receive feedback.
6. Succeed, fail, or retry.

## Validation

The loop is valid only when a player can complete it in Unreal without relying on external explanation.
