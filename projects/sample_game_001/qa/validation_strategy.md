# Sample Validation Strategy

## Purpose

Validation proves that a version is playable and meets its stated design goal.

## Checks

| Check | Purpose | Tool / Method |
|---|---|---|
| Map check | Catch Unreal map errors | GameDevMCP or Unreal Editor |
| Blueprint compile | Catch asset logic errors | GameDevMCP or Unreal Editor |
| Manual play test | Validate the loop | Human or agent-guided test |
| Design questions | Validate the intended experience | Review checklist |

## Result Format

Record pass/fail, evidence, issues, and next actions.
