# 버전 문서

이 디렉터리는 현재 프로젝트 상태가 어떤 플레이 가능 버전인지 정의한다.

버전 문서는 기능 목록이 아니라 플레이어에게 약속하는 경험과 검증 기준을 중심으로 작성한다.

## 반드시 들어갈 내용

- 버전의 의미
- 플레이어에게 약속하는 경험
- 포함되는 디자인 범위
- 필요한 시스템
- 필요한 제작 산출물
- 제외 범위
- 버전 검증 기준
- 알려진 문제
- 다음 버전 후보

## 상태

| 상태 | 의미 |
|---|---|
| `draft` | 방향과 포함 범위를 정의하는 중 |
| `assembling` | 시스템, 작업 항목, 세로 단면을 제작 중 |
| `first_playable` | 짧은 플레이 반복 구조가 동작함 |
| `validation_passed` | 버전 검증 기준 통과 |
| `released` | 내부 또는 외부 배포 완료 |
