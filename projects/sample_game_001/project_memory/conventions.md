# Sample Project Conventions

## Naming

- Use stable prefixes for project-owned assets.
- Do not use standalone prefix names as asset names.
- Keep Unreal paths explicit in asset manifests.

## Documentation

- Markdown is the main human review format.
- JSON is used only for automation or schema validation.
- Record decisions in `project_memory/` when they affect future work.

## Image References

- Keep generated images under category-local `images/<stage>/` folders.
- Mark replaced references as legacy or superseded.
