# 핵심 루프 시스템

## 목적

플레이어가 짧은 코스를 반복해서 플레이하고, 수집량과 클리어 시간을 개선하게 만든다.

## 상태 흐름

| 상태 | 설명 | 다음 상태 |
|---|---|---|
| Ready | 시작 대기 또는 리스폰 직후 | Playing |
| Playing | 이동, 점프, 수집, 회피 진행 | Cleared 또는 Failed |
| Cleared | 골 게이트 도달 | Result |
| Failed | 낙하, 충돌, 시간 초과 | Retry |
| Result | 기록 표시 | Ready |
