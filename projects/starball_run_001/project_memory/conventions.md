# Project Conventions

## Asset Prefixes

| Prefix | Meaning |
|---|---|
| `BP_Starball` | v0.2 prototype Blueprint assets |
| `BP_SR_` | v0.3 Starball Run gameplay object Blueprints |
| `SM_SR_` | Starball Run static meshes |
| `MI_SR_` | Starball Run material instances |
| `NS_SR_` | Starball Run Niagara systems |
| `SBR_*` | Actor label convention, not an Unreal asset name |

## Documentation

- Markdown is the authoritative human review format.
- JSON is used only when automation needs structured data.
- New asset names must include a definition, path, role, and validation target.

## Image References

- Concept references live under `images/concept_reference/`.
- Modeling references live under `images/modeling_reference/`.
- Replaced references move to `legacy/` or `superseded_reference/`.
