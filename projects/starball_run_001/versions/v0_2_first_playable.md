# v0.2 첫 플레이어블

## 목표

작은 플랫폼 게임으로서 시작, 이동, 수집, 회피, 클리어가 하나의 플레이 흐름으로 연결된다.

## 완료 기준

| 영역 | 기준 |
|---|---|
| 레벨 | 30~60초 코스가 있다 |
| 조작 | 이동과 점프로 진행 가능하다 |
| 수집 | 별/공 수집이 확인된다 |
| 장애물 | 충돌 또는 낙하 실패가 있다 |
| 클리어 | 골 게이트 도달 판정이 있다 |
| 검증 | 맵 체크 치명 오류 없음 |
| 아트 준비 | 전용 아트 제작 전략과 콘셉트 브리프가 있다 |

## 미완료 허용 범위

- 최종 아트
- 사운드
- 결과 UI
- 최고 기록 저장

최종 아트는 제외하지만, 아트 제작 계획과 대표 에셋 브리프는 제외하지 않는다.

## 현재 판정

조건부 진행 중.

## 현재 충족된 항목

- `Variant_SideScrolling` 기반 첫 플레이어블 맵을 `/Game/StarballRun/Maps/Lvl_Starball_FirstPlayable`로 복제했다.
- StarballRun 전용 캐릭터, 게임모드, 플레이어 컨트롤러, 수집물, 움직이는 발판, 소프트 발판 블루프린트를 복제했다.
- 복제된 블루프린트 6개가 MCP `blueprint.compile`에서 모두 성공했다.
- 현재 열린 `/Game/ThirdPerson/Lvl_ThirdPerson` 월드에는 검증용 `SBR_*` 액터 19개를 배치했다.
- MCP `world.validation.map_check` 결과 error 0, warning 18이다.
- 아트 전용 제작 전략과 콘셉트 브리프를 추가했다.

## 남은 항목

- StarballRun 전용 맵을 에디터에서 열고, 검증용 배치를 해당 맵 기준으로 다시 고정해야 한다.
- 골 게이트 클리어 로직과 장애물 실패 로직은 아직 전용 StarballRun 블루프린트로 구현되지 않았다.
- 수집물/장애물/골 게이트의 실제 전용 proxy 에셋은 아직 제작되지 않았다.
- 점수/시간 HUD는 v0.3 범위이므로 아직 제외한다.
