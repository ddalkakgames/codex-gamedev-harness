# Environment Concept

## Theme

Starball Run takes place in the upper levels of a celestial workshop: pale stone platforms, blue metal rails, gold astronomical devices, floating mechanisms, and soft sky depth.

## Layering

| Layer | Purpose | Rule |
|---|---|---|
| Gameplay layer | Platforms, collectibles, hazards, goal | Highest readability |
| Midground | Workshop machinery and rails | Supports direction without hiding gameplay |
| Background | Sky, stars, distant structures | Low contrast, atmospheric |

## Rule

Objects that affect gameplay are owned by `art/gameplay_objects/`. The environment document may link to them, but it should not redefine platform mechanics or hazard behavior.
