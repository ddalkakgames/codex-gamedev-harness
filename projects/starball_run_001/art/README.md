# Starball Run Art Harness

This folder owns art direction, generated references, asset briefs, art reviews, and art work items for Starball Run.

## Art Goal

Starball Run should feel like a bright celestial workshop: readable, toy-like, energetic, and clean enough for fast platformer play.

## Folder Map

| Folder | Purpose |
|---|---|
| `direction/` | Overall visual direction |
| `concepts/` | Concept briefs and overview images |
| `characters/` | Lumi character concept and motion direction |
| `gameplay_objects/` | Collectibles, hazards, goal gate, platforms, and lift system |
| `environment/` | Celestial workshop environment direction |
| `animation_vfx/` | Motion and VFX rules |
| `production/` | Asset briefs and production strategy |
| `reviews/` | Art review decisions |
| `work_items/` | Art work items |

## Image Storage

Images are stored under the category that uses them.

Examples:

- `characters/images/concept_reference/character_sheet_v001.png`
- `concepts/images/concept_reference/concept_overview_v002.png`
- `gameplay_objects/images/modeling_reference/static_meshes/sm_sr_star_shard_a_orthographic_v001.png`

## Rule

Generated images are references. Actual game assets must still be modeled, optimized, imported, reviewed, and validated in Unreal.
