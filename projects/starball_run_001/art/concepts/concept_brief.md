# 콘셉트 브리프

## 목적

`별공 런`의 전용 아트가 같은 방향으로 제작되도록 주인공, 수집물, 장애물, 골, 발판, 배경의 핵심 컨셉을 고정한다.

이 문서는 imagegen, 원화, 모델링, 레벨 아트 작업의 공통 기준이다. imagegen 프롬프트 자체는 문서에 노출하지 않고, 문서에는 이미지와 검토 결과만 남긴다.

## 한 줄 컨셉

작은 별 수리공 `루미`가 하늘 위 천문 공방을 달리며 흩어진 별 조각과 에너지 공을 회수하고, 별빛 골 게이트를 다시 켠다.

`루미`는 임시명이다. 이름은 바뀔 수 있지만, “작고 민첩한 별 수리공”이라는 역할과 실루엣 기준은 유지한다.

## 핵심 시각 목표

| 목표 | 설명 |
|---|---|
| 플레이어가 작고 민첩해 보여야 한다 | 캐릭터는 큰 전사나 마스코트가 아니라, 공방을 뛰어다니는 작은 수리공이다 |
| 별/공/위험/골이 즉시 구분되어야 한다 | 색상만이 아니라 형태, 발광, 움직임이 모두 달라야 한다 |
| 배경은 예쁘지만 조용해야 한다 | 천문 공방의 분위기는 주되, 발판과 위험물보다 눈에 띄면 안 된다 |
| 외부 IP를 연상시키면 안 된다 | 특정 레이드, 광대, 보스, UI, 캐릭터 실루엣을 가져오지 않는다 |

## 주인공 요약

| 항목 | 기준 |
|---|---|
| 이름 | 루미, 임시명 |
| 역할 | 별 에너지를 수리하는 어린 공방 기술자 |
| 체형 | 2.5등신 정도의 작은 인간형 또는 정령형 |
| 얼굴 | 큰 눈 2개와 단순 표정, 복잡한 얼굴 디테일 없음 |
| 머리/헬멧 | 별 모양 장식이 달린 둥근 작업 헬멧 또는 후드 |
| 의상 | 짧은 작업복, 작은 공구 벨트, 장갑, 큰 부츠 |
| 색상 | 짙은 남색 작업복, 크림색 장갑/부츠, 금색 별 장식, 청록색 공구 빛 |
| 금지 | 별 모양 그 자체의 캐릭터, 광대, 기사 갑옷, 마법사 로브, 기존 캐릭터 복제 |

자세한 기준은 `../characters/lumi_character_concept.md`를 따른다.

## 추가 이미지 요청 기준

추가 이미지가 필요하면 아래 기준을 우선한다. Codex가 내부적으로 imagegen을 사용하고, 문서에는 결과 이미지만 첨부한다.

| 이미지 | 목적 | 반드시 포함 | 반드시 피할 것 |
|---|---|---|---|
| 캐릭터 수정 시트 | 루미의 형태 수정 | 작은 인간형/정령형 수리공, 헬멧, 장갑, 부츠, 공구 벨트 | 별 자체 캐릭터, 광대, 기사, 마법사 |
| 스테이지 화면 | 첫 스테이지 분위기 확인 | 루미, 발판 경로, 별 조각, 에너지 공, 위험 장치, 골 게이트 | UI, 텍스트, 과도한 배경 장식 |
| 오브젝트 시트 | 게임플레이 오브젝트 형태 확인 | 별 조각, 에너지 공, 위험 장치, 골 게이트, 발판 키트 | 서로 비슷한 실루엣, 출처가 떠오르는 디자인 |
| 환경 키 | 천문 공방 배경 검토 | 낮은 대비 배경, 망원경/기어/별지도, parallax 층 | 플레이 경로보다 눈에 띄는 배경 |

## 스타일 제한

- 기존 상용 게임 캐릭터, 보스, 스테이지, UI, 고유 실루엣을 직접 연상시키면 안 된다.
- 주인공은 별 모양 생명체가 아니라 별 수리공이다.
- 위험물은 빨강/주황 계열이지만 공포스럽거나 잔혹하면 안 된다.
- 배경은 장식보다 플레이 판독성을 우선한다.
- 수집물과 위험물은 색맹 접근성을 고려해 색뿐 아니라 형태도 달라야 한다.
