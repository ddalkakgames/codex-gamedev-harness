# UX 설계

## 화면 정보

v0.2에서는 화면 메시지와 로그로만 검증하고, v0.3부터 HUD를 만든다.

## HUD v0.3 구성

| 위치 | 정보 |
|---|---|
| 좌상단 | 별 조각 수, 에너지 공 수 |
| 우상단 | 경과 시간 |
| 중앙 | 클리어/실패 짧은 메시지 |

## 피드백 기준

- 별 조각 획득: 작고 빠른 발광 효과
- 에너지 공 획득: 별보다 큰 효과와 다른 색
- 장애물 충돌: 즉시 정지 또는 리스폰 전 짧은 피드백
- 골 도달: 게이트 발광과 결과 메시지

## 입력 기준

입력은 기존 사이드 스크롤링 템플릿의 이동/점프를 우선 사용한다. 별도 입력 매핑 변경은 첫 플레이어블 검증 뒤 진행한다.
