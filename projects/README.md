# 게임 프로젝트 목록

이 디렉터리는 GameDevFramework로 운영하는 게임 프로젝트들을 보관한다.

각 프로젝트는 독립된 문서 루트를 가진다.

```text
projects/
  <project_id>/
    design/
    systems/
    art/
      README.md
      direction/
      concepts/
      characters/
      gameplay_objects/
      environment/
      animation_vfx/
      production/
      reviews/
      work_items/
    engineering/
    ux/
    qa/
    production/
    slices/
    versions/
    work_items/
    project_memory/
```
